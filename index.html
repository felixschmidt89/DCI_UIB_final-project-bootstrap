About me

I've spent my professional life (7 years) in different roles focusing on WEB software at dynamic, fast-growing and ever-changing tech startups. In my recent role as POS expert (4.5 years) at Treatwell's product team, it was my mission to lay the ground for gaining and retaining legal compliance of a Point of Sale software module in 10 different western European markets. In essence, I researched the vast variety of legal requirements in a market, broke them down and owned meeting all non-technical ones while at the same time conveying technical requirements to the dev team and closely consulting them on delivery. Becoming the go-to-guy for all newly emerged POS related topics and queries and occasionally stepping in as Product Manager, I acquired special knowledge about fiscalisation across Europe, slowly realizing that I highly enjoy the collaborated work and its constant challenges and I'd rather acquire generic web development skills to continue working in this field without being bound to a single - let's face it - rather uninspiring topic.

Consequently, and since I'm a quick learner, have proven to be a problem solver, good at mathematics and logics, excel at hours of focused, autonomous & challenging work paying great attention to detail and can tell from experience, that software development is where I want to work, I had decided to aim for a career change: First go on a sabbatical year and then move from product to tech and acquire what I'm lacking - coding skills - to not only work with web developers but become one myself.

About this project

After having built this website from scratch I've decided to create a second version of it - this time styled using bootstrap. This is to showcase that besides having learned how to design & build websites HTML, CSS and Sass I've also learned the basics of Bootstrap during the very first technical module ('User Interface Basic') as part of the Web Development Course at DCI. If you'd like to see my code, please refer to this Github repository. If you want to learn more about what other skills I'm about to acquire during the course at DCI, please give this article a read. Next up will be JavaScript & React.
